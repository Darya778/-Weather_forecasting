# Разработка модели для вероятностной оценки критических погодных условий с механизмом инкрементального обучения на локальных метеоданных

## Описание проекта

Проект представляет собой систему прогнозирования критических погодных условий на основе локальных метеорологических измерений.

Основная задача системы — оценка вероятности возникновения опасных погодных ситуаций, способных негативно повлиять на работу научного оборудования Института солнечно-земной физики СО РАН.

В качестве основы используется многозадачная модель Transformer, которая одновременно:

* прогнозирует погодные параметры;
* вычисляет интегральную оценку риска;
* определяет вероятность возникновения критических погодных условий;
* поддерживает инкрементальное обучение при поступлении новых данных.


## Используемые данные

В работе используются локальные метеорологические данные комплекса МАМКА (Мачтовый автоматический метеорологический комплекс астрономический).

Основные параметры:

* температура воздуха;
* влажность воздуха;
* скорость ветра;
* атмосферное давление;
* количество осадков;
* концентрация PM10;
* концентрация PM2.5.

Данные представлены в виде многомерных временных рядов с минутным разрешением.

## Основные возможности

### Предобработка данных

Система выполняет:

* очистку пропусков;
* удаление выбросов;
* интерполяцию значений;
* нормализацию признаков;
* формирование временных окон;
* генерацию дополнительных признаков.

### Прогнозирование

Модель прогнозирует:

* температуру;
* скорость ветра;
* влажность;
* осадки;
* концентрацию PM10;
* концентрацию PM2.5.

### Оценка риска

На основе прогнозируемых параметров рассчитывается интегральный риск возникновения критических погодных условий.

### Инкрементальное обучение

Это позволяет дообучать модель без полного переобучения и снижать эффект катастрофического забывания.
